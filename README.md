# federated-iam-observability-lab

End-to-end observability of federated identity infrastructure on AKS.

## Stack
- AKS (Azure Kubernetes Service)
- Keycloak 19 — Identity Provider (OIDC + SAML)
- FastAPI — protected microservices
- OpenTelemetry — instrumentation
- Prometheus + Grafana — metrics and dashboards
- Azure Monitor — cloud-native telemetry backend
